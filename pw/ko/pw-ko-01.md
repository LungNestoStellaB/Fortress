# PW Episode 01 — 당신이 이미 가진 씨앗

## Title
당신이 이미 가진 씨앗

## Description
대부분의 사람들은 완벽한 순간, 완벽한 도구, 완벽한 조건을 기다립니다. 부미볼 국왕은 결코 그러지 않았습니다. 그는 이미 있는 것으로 시작했습니다. 이 에피소드는 더 나은 삶의 씨앗이 이미 당신의 손에 있다는 것을 상기시킵니다.

## Clean Script

세상은 당신에게 말합니다: 기다려라.
돈이 더 많아질 때까지 기다려라.
공간이 더 생길 때까지 기다려라.
조건이 완벽해질 때까지 기다려라.

부미볼 국왕은 태국에서 가장 어려운 마을로 걸어 들어갔고
기다리지 않았습니다.
무릎을 꿇고 흙을 집어 들고 물었습니다:
"여기서 무엇을 키울 수 있을까?"

무엇이 부족한지가 아닙니다.
이미 여기 있는 것이 무엇인지.

하나의 씨앗. 하나의 땅. 시작하기로 한 하나의 결정.

자족은 모든 것이 준비되었을 때 시작되지 않습니다.
기다림을 멈추는 순간 시작됩니다.

당신은 이미 씨앗을 가지고 있습니다.
심으세요.
